# To connect to MSSQL Server you need to changing the port to 1433
# Open SQL Server Configuration Manager
# Choose SQL Server Network Configuration
# Choose protocol for ("localhost")
# Note: localhost can be difference
# Choose TCP/IP
# Switch to IP addresses tab, scroll to the end, at the IPALL expand 
# Type 1433 at TCP port
